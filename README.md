# FizzBuzz

This is Fizz Buzz application, built with javascript. Jest is used for test. 

## Installation and Setup Instructions

#### Example:  

Clone down this repository. You will need `node` and `npm` installed globally on your machine.  

Installation:

`npm install`  

To Build App:

`npm build`

To Run Test Suite:  

`npm test`  

To Start Server:

`npm start`  

To Visit App:

`localhost:8080`
